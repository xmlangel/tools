# 프로젝트 규칙 (Project Rules)

## 문서 작성 규칙 (Documentation Rules)
1. **언어**: 모든 산출물(Walkthrough, Implementation Plan, Task List 등)은 **반드시 한글**로 작성해야 합니다.
2. **커밋 메시지**: 커밋 메시지는 영문으로 작성하되, 명확하고 간결하게 작성합니다.

## 코드 작성 규칙 (Code Authoring Rules)
1. **파일 크기 제한**: 각 파일은 1000 라인을 초과하지 않도록 관리합니다.
2. **모듈화**: 기능 및 의미 단위로 파일을 분리하여 유지보수가 용이하도록 구조화합니다.
